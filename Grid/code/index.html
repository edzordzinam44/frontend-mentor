A bit about me

I'm a BTech student at Koforidua Technical University, currently training as a frontend developer through ALX Ghana. Most days you'll find me deep in a browser tab, rebuilding a layout I just learned about, or arguing with CSS Grid until it finally listens.

I work mainly with HTML, CSS, JavaScript and React.js, and I like projects that force me to actually think through layout and structure — not just copy a template. Recent builds include a bento-grid landing page (Frontend Mentor challenge) and a recipe page styled from scratch, both written by hand in HTML/CSS without a framework, just to keep the fundamentals sharp.

I care about writing code that's readable before it's clever, and I'm always one tutorial away from learning something that breaks my whole layout — in a good way.
